# 会員制動画サイトの使い方

このフォルダには、既存サイトに加えて「会員ログイン」「会員専用動画ページ」「管理者の動画アップロード画面」が入っています。

## 初回セットアップ

1. `.env.example` をコピーして `.env` を作ります。
2. `.env` の `ADMIN_EMAIL` と `ADMIN_PASSWORD` を自分用の管理者ログインに変更します。
3. ターミナルで以下を実行します。

```powershell
npm install
npm start
```

4. ブラウザで `http://localhost:3000/login` を開きます。
5. `.env` に書いた管理者メールアドレスとパスワードでログインします。

`.env` をまだ作っていない場合は、初回起動時に `data/initial-admin-login.txt` が作られます。そこにログインURL、管理者メールアドレス、初期パスワードが書かれています。

## 動画をアップロードする流れ

1. 動画を収録します。
2. 可能なら `mp4` 形式で書き出します。
3. 管理画面 `http://localhost:3000/admin` を開きます。
4. 「動画をアップロード」からタイトル、分類、説明、動画ファイルを入力します。
5. 「すぐ公開する」を選んでアップロードすると、会員ページ `http://localhost:3000/library` に表示されます。

対応ファイルは `mp4`, `mov`, `m4v`, `webm` です。アップロード上限は1本あたり約1.5GBです。

## 会員を追加する流れ

1. 管理画面 `http://localhost:3000/admin` を開きます。
2. 「会員を追加」に名前、メールアドレス、初期パスワードを入力します。
3. 生徒には `http://localhost:3000/login` とログイン情報を伝えます。

## データの保存場所

- 会員情報: `data/users.json`
- 動画情報: `data/videos.json`
- 動画ファイル: `uploads/videos/`

`data/` と `uploads/` は `.gitignore` に入れてあります。公開リポジトリには個人情報や動画を入れないでください。

## 本番公開するときの注意

本番では必ず `.env` の `SESSION_SECRET` を長いランダム文字列に変更してください。

動画ファイルを大量に置く場合は、サーバー本体ではなくS3などのストレージに置く構成へ拡張するのがおすすめです。まずはこのローカル保存版で運用を始められます。
